Laser Work

During my last two years of high school, I was introduced to a laser cutter and immediately was intrigued and created many designs for friends and family for different keychains at the time.

Skateboard Design

This is an engraved skateboard design project I did recently in which I had to create the design in either Illustrator, which I chose, or Rhino, making sure the stroke colors were the exact RGB values that the laser engraver would read, and then send it to get engraved.

Flatfab Cut

We had to learn to use this software called Flatfab in which you can create shapes with interconnected pieces and the program would have those cuts already created in the file for you to be able to cut without guess and checking.
